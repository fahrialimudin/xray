Jalankan perintah berikut di VPS Anda sebagai root:

cd /root
wget https://github.com/fahrialimudin/xray/raw/main/panel-simpel.zip
unzip -o panel-simpel.zip
cd panel-simpel
chmod +x install-panel.sh
bash install-panel.sh


Yang akan dilakukan script ini:
Install PHP 8.3 + PHP-FPM
Deploy file panel ke /home/vps/public_html/
Setup sudoers untuk www-data
Konfigurasi Nginx di port 1313
Reload Nginx & test API


Setelah install berhasil:
Akses panel: http://IP_VPS:1313
